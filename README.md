# Wendesen Kebede - Full-Stack Developer Portfolio

A modern, responsive portfolio website built with React, Vite, and Tailwind CSS.

## Features

- **Modern React Architecture**: Built with React 18 and Vite for fast development
- **Beautiful UI**: Styled with Tailwind CSS for a clean, professional look
- **Responsive Design**: Fully responsive across all devices
- **Smooth Scrolling**: Navigation with smooth scroll behavior
- **Contact Form**: Functional contact form that sends messages via Google Apps Script
- **Toast Notifications**: User-friendly feedback system
- **Component-Based**: Modular React components for easy maintenance

## Sections

- **Home**: Hero section with introduction and call-to-action
- **About**: Personal information and career objectives
- **Education**: Academic background and relevant coursework
- **Skills**: Technical skills with progress bars
- **Projects**: Featured projects with technology stack
- **Contact**: Contact form and direct contact information

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Preview Production Build

```bash
npm run preview
```

## Technologies Used

- **React 18**: UI library
- **Vite**: Build tool and dev server
- **Tailwind CSS**: Utility-first CSS framework
- **Font Awesome**: Icon library
- **Google Fonts**: Inter font family

## Contact Form

The contact form uses a Google Apps Script to send messages directly to the email address. The script URL is configured in the Contact component.

## Customization

To customize the portfolio for your own use:

1. Update personal information in each component
2. Replace the profile image URL in Hero.jsx
3. Update the Google Apps Script URL in Contact.jsx
4. Modify the color scheme in Tailwind classes
5. Add or remove projects in Projects.jsx
6. Update skills in Skills.jsx
